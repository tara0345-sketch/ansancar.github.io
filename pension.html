펜션 예약 신청

안내사항 및 동의

1. 예약

1.1 이용금액 : 개인 - 10만원, 구역 및 교구(소모임 포함) - 5만원, 타교회 - 10만원

2. 이용

2.1 비밀번호 : 예약 확정 시 펜션 비밀번호 안내 (펜션 비밀번호는 이용자 외 외부에 공유 금지)

2.2 입·퇴실 시간 : 입실 15:00 / 퇴실 15:00 (청소 완료 후 퇴실 시간)
(입/퇴실 시간 관련 특별한 사유 발생 시 담당자에게 문의)
* 담당자 연락처: 010-9200-1199 (김선도B)

2.3 청소 : 입실 시와 동일한 컨디션으로 청소

2.4 비품 :

  • 1) 퇴실 시 비품은 기존 상태 그대로 유지할 수 있도록, 가져온 비품(ex: 수건, 치약 등)은 빠짐없이 모두 챙겨갈 것
  • 2) 퇴실 시 비품의 부족 및 손상 시 신속하게 담당자에게 내용 전달

3. 퇴실 전 체크리스트

  • 3.1 설거지 및 청소 상태 확인 (청소기 바닥 먼지 제거, 바닥 닦기 등 / 입실 시와 동일 컨디션)
  • 3.2 쓰레기 분출
    • ① 일반 쓰레기 : 수양관 내 분리수거장 (약도첨부)
    • ② 음식물 쓰레기 : 밤나무 옆 (약도첨부)
  • 3.3 펜션 내의 모든 조명 전원 off (현관 일괄소등 스위치 off)
  • 3.4 퇴실 전 냉방기 전원 off 및 난방기 외출 모드
  • 3.5 가스밸브 잠김 확인
  • 3.6 기타 가전 (선풍기 등) 전원 코드 뽑기
  • 3.7 사진 촬영 후 담당자에게 전송

해당사항을 확인하신 후 아래 체크박스에 체크하고 제출 버튼을 눌러주시면 신청이 접수됩니다.
